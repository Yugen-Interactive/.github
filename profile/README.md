# Yūgen Interactive — Employee README

## About Yūgen Interactive

Yūgen Interactive is an independent game development studio focused on creating unique and experimental gaming experiences. The studio aims to build immersive and memorable games that stand out in the indie scene.

---

## Philosophy

* Creativity over trends
* Player experience is the priority
* Small team, high impact
* Fast iteration and continuous improvement

---

## Team Structure

Roles may overlap. Collaboration is expected across all areas.

Core roles:

* Game Design
* Programming (Gameplay, Systems, Tools)
* Art (2D / 3D)
* Sound Design
* QA Testing

---

## Tech Stack

Depends on the project:

* Engines: Godot, Unity, Unreal
* Languages: GDScript, C#, C++
* Version Control: Git
* Project Management: Notion, Trello, Jira
* Communication: Discord, Slack

---

## Workflow

1. Concept
2. Prototype
3. Internal testing
4. Iteration
5. Polish
6. Release

Focus on speed, but prioritize quality and game feel.

---

## Coding Guidelines

* Write clean and modular code
* Use clear and descriptive names
* Avoid unnecessary complexity
* Keep code readable
* Follow project architecture

---

## Art Guidelines

* Maintain visual consistency
* Optimize assets for performance
* Reuse assets when possible
* Coordinate with design and programming

---

## Testing and QA

* Test continuously
* Report bugs with:

  * Steps to reproduce
  * Expected result
  * Actual result
  * Evidence (screenshots or video)

---

## Communication

* Be clear and direct
* Share progress regularly
* Ask for help when needed
* Provide constructive feedback

---

## Work Culture

* Flexible and asynchronous-friendly
* Responsibility over assigned tasks
* Ownership mindset
* Results-driven

---

## Security

* Do not share internal content externally
* Use version control correctly
* Keep backups of important work

---

## Growth

* Learn new tools and technologies
* Experiment and propose ideas
* Contribute improvements
* Take initiative

---

## Final Notes

Every team member contributes to the final product. Initiative, creativity, and responsibility are expected at all times.
